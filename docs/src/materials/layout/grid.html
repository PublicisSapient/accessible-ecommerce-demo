--- notes: | For the 'grid', instead of using an overly-complex method of using wrapper/presentation class divs with no semantic value. This approach uses flexbox and is implemented by way of using a mixin on the 'parent' container that will determine the layout for the 'children'. The mixin takes two parameters: - the first parameter can be either 'auto' or a number of columns - the second parameter is a boolean and is used determine the use of gutters. Defaults to true. For a responsive grid, which changes columns at lower breakpoints, use media queries. ```css .auto-columns-no-gutter-example{ @include flex-grid('auto', false); } .auto-columns-with-gutter-example{ @include flex-grid('auto'); } .fixed-grid-4-columns-no-gutter-example{ @include flex-grid(4, false); } .fixed-grid-4-columns-with-gutter-example{ @include flex-grid(4); } ``` ---

Forms

Checkbox

The real magic of the styled form components is mostly in the CSS:

  • The input element is given opacity: 0 so that it can't be seen, but is still an active element on the page.
  • The label and the input are connected via for and id attributes, which allows the input to be triggered by clicking on the label
  • The new checkbox UI is created with the ::before and ::after psuedo elements on the label
  • The :focus and :checked states on the input are passed to the label with the sibling selector (input:checked + label), so it important to note that the input element must preceed the label.

Accessibility:

Groups of checkbox inputs should be wrapped in a fieldset and have a legend, which can be hidden from sighted users.

Filter by Colors 


<fieldset>
	<legend tabindex="0" id="colorsLegend" class="visibly-hidden">Filter by Colors</legend>
	<input id="blue" class="checkbox__input" aria-label="Filter on color blue" type="checkbox" name="colors" value="blue">
	<label for="blue" class="checkbox__label">Blue</label>
	<br>
	<input id="green" class="checkbox__input" aria-label="Filter on color green" type="checkbox" name="colors" value="green">
	<label for="green" class="checkbox__label">Green</label>
	<br>
	<input id="yellow" class="checkbox__input" aria-label="Filter on color yellow" type="checkbox" name="colors" value="yellow">
	<label for="yellow" class="checkbox__label">Yellow</label>
	<br>
	<input id="red" class="checkbox__input" aria-label="Filter on color red" type="checkbox" name="colors" value="red">
	<label for="red" class="checkbox__label">Red</label>
</fieldset>

Color Selection

	<legend id="color">Color Selection</legend>
	<input type="radio" class="radio__input" value="red" name="color" id="color-red" aria-describedby="color">
	<label for="color-red" class="radio__label"> Red</label>
	<input type="radio" class="radio__input" value="orange" name="color" id="color-orange" aria-describedby="color">
	<label for="color-orange" class="radio__label">Orange</label>
	<input type="radio" class="radio__input" value="white" name="color" id="color-white" aria-describedby="color" checked>
	<label for="color-white" class="radio__label">White</label>
</fieldset>

Elements

Buttons

Primary Button

The default button is 48px high and uses the primary colour for the background. The .btn classes can be applied to <a> tags to create links that have the same appearance as a button.

Primary Link 
<button class="btn">Primary Button</button>
<a class="btn">Primary Link</a>

Variations

Different button styles are possible by adding additional styling classes. Note: There is no defined styling for an outlined button that is disabled

<button class="btn btn--black">Black</button>
<button class="btn btn--outline">Outline</button>
<button class="btn" disabled>Disabled</button>
<button class="btn btn--icon"><span class="icon ion-ios-heart"></span></button>
<button class="btn btn--outline btn--icon"><span class="icon ion-ios-add"></span></button>

Typography

The typography for the site is using the 62.5% font size on the html with rem units for the rest of the site. This approach allows for easier mental math to create root-relative font sizes, while also allowing users to change their default font sizes (because the reset is a % and not fixed px). The 62.5% takes the general browser default of 16px and sets it to 10px. The body has been set to 1.6rem which translates back to 16px. For users that change their default browser font size, the relative size differences between the various components and elements will be preserved.
